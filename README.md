# E-Book Cover AI 利用マニュアル

## 1. はじめに
このアプリは、Googleの最新AI「Gemini」を利用して、誰でも簡単にプロ品質の電子書籍表紙を作成できるツールです。
日本語の縦書き、帯（オビ）の作成、レイアウト調整など、Kindle出版に必要な機能を網羅しています。

---

## 2. 利用開始までの準備（初回のみ）

本アプリを使用するには、Googleが発行する「APIキー（無料）」が必要です。
以下の手順で取得し、アプリに設定してください。

### 手順①：APIキーの取得
1. **[Google AI Studio](https://aistudio.google.com/app/apikey)** にアクセスします。
2. Googleアカウントでログインします。
3. 左上の **「Get API key」** をクリックします。
4. **「Create API key」** ボタンを押します。
   （プロジェクトの選択画面が出た場合は「Create API key in new project」を選択してください）
5. 生成された `AIza` から始まる長い文字列をコピーしてください。

### 手順②：アプリへの設定
1. アプリを起動します。
2. 初回起動時に「APIキーの設定」画面が表示されます。
   （表示されない場合は、画面右上の「⚙️（歯車マーク）」をクリックしてください）
3. コピーしたAPIキーを入力欄に貼り付け、**「設定を保存」** をクリックします。

※設定したキーはあなたのブラウザ内（Local Storage）にのみ保存され、外部に送信されることはありません。

## 3. 基本的な使い方

### ① 作品情報の入力
画面左側のフォームに、本の情報を入力します。

- **カテゴリー**: 「小説」「ビジネス」「マンガ」などから選択します。
- **魔法の杖ボタン（おすすめ設定）**: カテゴリーを選んで「～のおすすめ設定を適用」ボタンを押すと、そのジャンルで「売れる」配色やフォントが自動セットされます。迷ったらまずはこれを押しましょう。
- **あらすじ**: AIはここを読んで絵を描きます。できるだけ詳しく書くと、イメージ通りの絵になりやすいです。

### ② 表紙を生成する
一番下の「表紙を生成する」ボタンを押します。15秒～30秒ほどお待ちください。

### ③ 編集・カスタマイズ（重要）
生成された画像は、右側のプレビュー画面で直感的に編集できます。

- **文字の移動**: タイトル、著者名、帯、バッジなどは、**マウスでドラッグ**して好きな場所に動かせます。
- **文字サイズ変更**: 文字の上にカーソルを置き、**マウスホイールを回す**と、拡大・縮小できます。
- **改行幅の調整**: 文字を選択した時に出る「ハンドル（丸い点）」をドラッグすると、改行の幅（ボックスサイズ）を調整できます。
- **部分装飾**: フォームの「タイトルロゴ設定」＞「部分装飾」タブを使うと、特定の文字（例：「の」）だけ色やフォントを変えられます。

### ④ 保存
「表紙を保存」ボタンを押すと、高解像度（1600 x 2560 px）のPNG画像としてダウンロードされます。
そのままKindle (KDP) の表紙として使用できます。

## 4. よくある質問 (FAQ)

**Q. APIキーを忘れました / 無効だと言われます**
A. キーの再発行が必要です。Google AI Studioでもう一度「Create API key」を押して新しいキーを取得し、アプリ右上の歯車マークから新しいキーを上書き保存してください。

**Q. 「画像が生成されませんでした」とエラーが出ます**
A. あらすじに過度な性的・暴力的な表現が含まれている場合、Googleの安全フィルターによりブロックされることがあります。表現を修正して再度お試しください。

**Q. 別のパソコンで使うには？**
A. APIキーはブラウザごとに保存されます。別のパソコンで使う場合は、そのパソコンでも同じ手順でAPIキーを設定してください（同じキーを使い回しても問題ありません）。

**Q. 文字の配置が保存した画像と少しズレます**
A. ブラウザの表示倍率（ズーム）が100%でない場合にズレることがあります。ブラウザのズームを100%に戻してから保存してください。
